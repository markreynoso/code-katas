# code-katas

Day of Code Wars Challenges

Code Fellows 401 project to complete Code Wars Katas in 12 hours. 
Each Kyu is ranked as follows:
- 8th kyu: 1 
- 7th kyu: 2 
- 6th kyu: 4 
- 5th kyu: 5 
- 4th kyu: 6 
- 3rd kyu: 8 
- 2nd kyu: 9 
- 1st kyu: 10
The goal is to complete 20pts worth of Katas with any combination of Katas.
